Guia de Início Rápido para GitHub

Este guia irá orientá-lo através dos passos essenciais para começar a usar o GitHub, desde a criação do seu primeiro repositório até o seu primeiro commit.

Nota: Este é um guia para iniciantes. Para tópicos avançados, consulte as seções específicas da documentação.

Criando um Repositório

Um repositório é como uma pasta para o seu projeto. O seu projeto pode conter qualquer tipo de arquivo, como código, imagens ou arquivos de áudio.

  1. No canto superior direito, use o menu suspenso e selecione Novo repositório.
  2. Digite um nome curto e memorável para o seu repositório. Por exemplo, "hello-world".
  3. Escolha a visibilidade do repositório (Público ou Privado).
  4. Selecione Inicializar este repositório com um README.
  5. Clique em Criar repositório.

Fazendo seu primeiro commit

Um commit é como um snapshot das alterações feitas nos seus arquivos. Fazer commits permite que você salve o seu trabalho em pontos específicos.

Editando o arquivo README

Para editar o arquivo, clique no ícone de lápis. No editor, escreva um pouco sobre você. Quando estiver satisfeito, escreva uma mensagem de commit que descreva as suas alterações.
